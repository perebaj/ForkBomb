# ForkBomb
Process and Thread Linux behavior

###### Running fork_bomb.c:
gcc -Wall fork_bomb.c -o  fork_bomb
./fork_bomb

###### Running thread_bomb.c:
gcc -Wall -pthread thread_bomb.c -o thread_bomb
./thread_bomb
